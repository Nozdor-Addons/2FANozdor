# 2FANozdor

Аддон для управления двухфакторной аутентификацией (2FA) в World of Warcraft. Автоматически захватывает ключ 2FA из системных сообщений игры, отображает его в удобном окне и генерирует QR-код для быстрой настройки в приложении-аутентификаторе. Позволяет активировать или отключить двухфакторную аутентификацию прямо из игры с помощью 6-значного кода.
